Welcome to Open3D Examples GUI Tutorial 
=======================================

This is a tutorial of a Python visualization GUI for Open3D.

Open3D comes with a Python example visualization GUI that is both powerful and versatile. This tutorial will walk you through the steps needed to install Open3D's Python package, clone the Open3D Python package GitHub, and start the example visualization GUI. Then, this tutorial will explain how to open a 3D model and manipulate it within the GUI. The screen captures for the tutorial are below the descriptions or at the top of the following page. The tutorial will not cover these model manipulations with Python, but it will briefly examine the python code for the GUI.
